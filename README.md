# Project Nuevo Mundo
=============

Temporary PNM landing page with basic JSON backend

Installation
------------

For email / bitrix submission define assets/php/auth.php

* $mail_host
* $mail_port
* $mail_ssl
* $mail_user
* $mail_pass
* $bitrix_user
* $bitrix_pass

For mailchimp submission define assets/php/chimpkey.php

* $chimpkey
* $listid
